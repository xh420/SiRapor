SiRapor (singkatan dari Sistem Informasi Rapor) adalah sebuah aplikasi desktop sederhana namun fungsional yang dibangun menggunakan bahasa pemrograman Python. Aplikasi ini dirancang untuk mendigitalkan proses rekapitulasi nilai siswa yang sebelumnya manual menjadi sistem komputerisasi yang otomatis.

Aplikasi ini menggunakan antarmuka grafis (GUI) berbasis Tkinter, sehingga pengguna (Guru/Admin) dapat berinteraksi melalui tombol, tabel, dan formulir, bukan lewat kode atau terminal.
